# Student-Study-Planner

INTRODUCTION:

Student Study Planner is an intelligent application written in Python that seeks to help students overcome the challenges of dealing with highly pressurizing academic situations. It is not just an ordinary calendar application, as it uses AI logic to provide the student with their own dynamic study plan based on their performance every day of the week.

PROJECT OVERVIEW:

The "Student Study Planner" is a system that utilizes AI to help a student manage his/her time table effectively. It is developed for the Fundamentals of AI and ML course of VIT, Bhopal. The objective of this project is to optimize time management for a student, so that there is less stress involved in exams. The primary objective of this project is to resolve the issue faced by a student in managing his/her time table in a way that there are subjects of different levels of difficulty. The time table of a student does not operate on a calendar-based system, as it does in general.

FEATURES:

  -  Adaptive Scheduling
   - Priority Calculation
   - Personalized Slots
   - Built-in Well-being
   - Performance Analytics

How to Use:
- Input Subjects: Enter the number of subjects, their difficulty levels, and the days left for each exam.
- Define Constraints: Set your total daily study hours and the duration of your planning period.
- Personalize: Input your learning speed (Fast/Slow) and your peak energy time (Morning/Evening).
- Daily Updates: At the end of each day, report if you completed the plan and your distraction level (1–10).
- Review & Adjust: The system will generate a "Day-wise Plan" showing allocated hours and specific time slots for each subject.
